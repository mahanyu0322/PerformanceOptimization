### PerformanceOptimization主要有两大模块
#### 模块一
主要功能:防止crash造成APP闪退
防止一下异常情况

* NSArray
	* arrayWithObjects:count:
	* objectsAtIndexes:
	* objectAtIndex:
	* objectAtIndexedSubscript:
	* getObjects:range:
* NSMutableArray
	* objectAtIndex:
	* objectAtIndexedSubscript:
	* setObject:atIndexedSubscript:
	* removeObjectAtIndex:
	* insertObject:atIndex:
	* getObjects:range:
